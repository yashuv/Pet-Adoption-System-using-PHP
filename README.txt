# Installation

1. Open your XAMPP Control Panel and start "Apache" and "MySQL".
2. copy and paste the source code folder named "ecommerce" into XAMPP's "htdocs" directory.
3. Open a web browser and browse the PHPMyAdmin. (http://localhost/phpmyadmin)
4. Create a new database naming "db_ecommerce".
5. Import the SQL file to your newly created database. The SQL file is located inside the extracted source code folder and known as "db_ecommerce.sql".
6. Open a web browser and browse the web application. (http://localhost/ecommerce) for the website and for the admin side (http://localhost/ecommerce/admin)
